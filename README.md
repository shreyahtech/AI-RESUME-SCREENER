# 🤖 AI Resume Screener Pro

An AI-powered recruitment platform that helps recruiters analyze resumes, identify top candidates, and streamline the hiring process using semantic matching and ATS evaluation.

---

## 🚀 Features

✅ Semantic Resume–Job Description Matching

✅ ATS Score Analysis

✅ Candidate Ranking & Comparison

✅ Skill Gap Detection

✅ AI-Generated Recruiter Feedback

✅ Interview Question Generation

✅ Interactive Analytics Dashboard

✅ Resume Parsing & Information Extraction

✅ CSV Export of Results

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Sentence Transformers (BERT-based Semantic Matching)
- Pandas
- Plotly
- PyPDF2
- Scikit-Learn
- HTML & CSS

---

## 📊 How It Works

1. Upload one or multiple resumes (PDF).
2. Paste the job description.
3. AI analyzes resumes using semantic similarity.
4. ATS score is calculated.
5. Candidates are ranked automatically.
6. Missing skills and strengths are identified.
7. Recruiters receive actionable insights and interview questions.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/AI-Resume-Screener-Pro.git
cd AI-Resume-Screener-Pro
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
